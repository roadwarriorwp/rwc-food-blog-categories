== Description ==

The Food Blog Categories plugin adds recipe related taxonomies to blog posts. Intended for Food Bloggers or Food Product Businesses that want to include recipes in their blog posts and need a better system of organization.

== Installation ==

1. **Visit** Plugins > Add New in the WordPress admin dashboard
2. **Search** for "Food Blog Categories"
3. **Install** and then **Activate** Food Blog Categories from your Plugins page
3. **Navigate** to the taxonomies and add your terms
